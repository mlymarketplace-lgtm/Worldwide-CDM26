# QualifGaïndé Worldwide — V10.3.3 cadrée

Objectif : améliorations UX ciblées sans toucher au moteur sportif.

## Ajouts
- Score live/final affiché dans la card compte à rebours quand l’API-Sports renvoie un match live ou terminé.
- Scores des 16es remontés en face des équipes dans chaque card, avant le bouton “Choisir”.
- Ambiance YouTube par équipe pour les pays fournis : Sénégal, France, Algérie, Maroc, Belgique, Brésil, Côte d’Ivoire, Égypte, RD Congo.
- Lisibilité renforcée : badges, cards 16es, récits d’épopées et teasers plus grands/gras.
- Compteur supporters réglé autour de 11 120.
- Service Worker versionné V10.3.3.

## Prudence
- Aucun changement sur scores.js, moteur de simulation, tableaux, stats.json ou données d’équipes.
- Les ambiances restent déclenchées par clic utilisateur pour respecter les règles navigateur anti-autoplay.
