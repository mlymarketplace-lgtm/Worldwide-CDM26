## Recette Netlify obligatoire après déploiement
1. Ouvrir `/admin-gaindes/` et se connecter avec `QG_GAINDES_ADMIN_PASSWORD`.
2. Charger l’exemple Nicolas Jackson.
3. Ajouter une image de moins de 3 Mo.
4. Enregistrer en brouillon : vérifier son absence sur le site public.
5. Prévisualiser dans la console.
6. Publier : vérifier son apparition dans les Brèves des Gaïndés et sur le portail home.
7. Modifier le bloc « Le regard de QualifGaïndé ».
8. Dépublier : vérifier son retrait public.
9. Republier puis effectuer un nouveau déploiement : vérifier sa persistance.
10. Exporter le JSON de sauvegarde.
